memex(1)                        HYPERTEXT TOOLS                        memex(1)

NAME
     memex - associative information storage and retrieval system

SYNOPSIS
     memex [options] [command] [arguments]

DESCRIPTION
     memex is a hypothetical electromechanical device for augmenting human 
     memory and knowledge management, as proposed by Vannevar Bush in 1945.
     
     it allows users to store, organize, and retrieve information through 
     associative indexing and trails of links between documents.

COMMANDS
     store       Add new information to the memex
     retrieve    Find and display stored information
     associate   Create links between pieces of information
     trail       Record and replay a sequence of associations

OPTIONS
     -h          Display this help information
     -v          Verbose output
     -t          Use text-only interface

EXAMPLES
     memex store "Hypertext concept invented by Ted Nelson"
     memex retrieve "hypertext"
     memex associate "Bush, Vannevar" "Nelson, Ted"

FILES
     ~/.memexrc   User configuration file
     /var/memex/  Data storage location

AUTHOR
     Vannevar Bush (conceptual)

SEE ALSO
     xanadu(1), www(1), wikis(7)
