NAME
     memex - augment human intellect through associative information storage

SYNOPSIS
     memex [OPTIONS] [COMMAND]

DESCRIPTION
     memex is a hypothetical electromechanical device for augmenting human 
     memory and knowledge management, as proposed by Vannevar Bush in 1945.
     It serves as an analog hypertext system to store, retrieve, and 
     connect information.

COMMANDS
     store        Add new information to the memex
     retrieve     Find and display stored information
     link         Create associative trails between pieces of information
     browse       Navigate through stored information and links
     annotate     Add notes and metadata to stored items

OPTIONS
     -m, --microfilm   Use microfilm for storage (default)
     -d, --digital     Use digital storage
     -v, --verbose     Increase output verbosity
     -h, --help        Display this help message

EXAMPLES
     memex store "As We May Think by Vannevar Bush"
     memex retrieve "hypertext"
     memex link "Memex" "World Wide Web"

SEE ALSO
     brain(1), hypercard(1), xanadu(1), www(1)

AUTHOR
