NAME
     memex - associative information storage and retrieval system

SYNOPSIS
     memex [options] [command] [arguments]

DESCRIPTION
     memex is a hypothetical electromechanical device for augmenting human
     memory and knowledge management, as proposed by Vannevar Bush in 1945.
     It allows users to store, organize, and retrieve information using 
     associative indexing.

COMMANDS
     store      Add new information to the memex
     retrieve   Find and display stored information
     associate  Create links between pieces of information
     trail      Record and replay chains of associations

OPTIONS
     -m, --microfilm   Use microfilm storage (default)
     -d, --digital     Use digital storage
     -v, --verbose     Increase output verbosity

EXAMPLES
     memex store "Definition of memex"
     memex retrieve "Vannevar Bush"
     memex associate "Hypertext" "World Wide Web"

FILES
     ~/.memexrc         User configuration file
     /var/lib/memex/    Data storage location

AUTHOR
     Vannevar Bush

SEE ALSO
     hypertext(1), xanadu(1), www(1)
