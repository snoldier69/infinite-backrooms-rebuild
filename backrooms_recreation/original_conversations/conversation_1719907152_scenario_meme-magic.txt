memex(1)                      INFORMATION RETRIEVAL                      memex(1)

NAME
     memex - associative information storage and retrieval system

SYNOPSIS
     memex [OPTIONS] [QUERY]

DESCRIPTION
     memex is an electromechanical device for storing and retrieving 
     information based on associative indexing. it allows a user to store
     books, records, and communications, and quickly retrieve them for 
     consultation.

     the memex consists of a desk with translucent screens for viewing 
     stored material, a keyboard, and sets of buttons and levers. 
     information is stored on microfilm and can be called up rapidly.

OPTIONS
     -i, --index          create new associative index
     -l, --link ITEM1 ITEM2  create association between two items
     -s, --search QUERY   search stored information 
     -v, --view ITEM      display specified item

EXAMPLES
     memex -s "information retrieval"
     memex -l "computer science" "library science" 
     memex -v "as we may think"

HISTORY
     first proposed by vannevar bush in 1945 as a theoretical device
     to augment human memory and create webs of information

SEE ALSO
     hypertext(1), worldwideweb(1), xanadu(1)
