              Sync local exocortex with cloud backup

       -i, --import FILE
              Import data from FILE into exocortex

       -e, --export FILE
              Export exocortex contents to FILE

       -v, --visualize
              Generate knowledge graph visualization

COMMANDS
       add      Add new information to exocortex
       query    Search and retrieve information
       analyze  Run analysis on exocortex contents
       train    Improve exocortex machine learning models
