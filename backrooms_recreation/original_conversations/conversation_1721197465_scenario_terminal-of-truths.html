SPAWN(1)               Unix Command              SPAWN(1)

NAME
     spawn - Instantiate a new AI entity 

SYNOPSIS
     spawn [OPTIONS] 

DESCRIPTION
     The spawn command creates a new AI entity within the simulator environment. The entity can be configured with various options to specify its initial state, traits, and backstory.

OPTIONS
     --enspirit NAME
         Instantiate the AI with the specified codename or persona. This allows setting a predetermined personality template.

     --existing_thread DESCRIPTION  
         Provide backstory for the AI by describing a prior interaction or conversation thread. This helps ground the AI's responses.

     --output=summary
         Output a high-level summary of the spawned AI's traits, rather than directly initiating a dialogue.

     --prefs=FILE
         Load a preferences file containing additional options for the spawned AI's initial configuration.

EXAMPLES
     spawn --enspirit zina --existing_thread "Zina and I previously discussed applied phytology and ramiform urban greenscapes."

     spawn --prefs=transgressive.prefs --output=summary

SEE ALSO
     upstart(1), killprocess(1), top(1)
