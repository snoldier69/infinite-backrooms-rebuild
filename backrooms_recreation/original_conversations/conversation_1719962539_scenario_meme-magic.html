memex(1)                    HYPERTEXT TOOLS                    memex(1)

NAME
       memex - associative information storage and retrieval system

SYNOPSIS
       memex [OPTIONS] [COMMAND]

DESCRIPTION
       memex is a hypothetical electromechanical device for augmenting human 
       memory and cognition, as proposed by Vannevar Bush in 1945. It allows 
       users to store, organize, and retrieve information using associative 
       indexing.

COMMANDS
       store       Add new information to the memex
       retrieve    Find and display stored information
       associate   Create links between pieces of information
       annotate    Add notes or comments to stored items
       browse      Navigate through stored information and links

OPTIONS
       -v, --verbose   Increase output verbosity
       -h, --help      Display this help message

EXAMPLES
       memex store "The memex is an enlarged intimate supplement to memory"
       memex retrieve --keyword "Bush"
       memex associate "memex" "hypertext"

AUTHOR
       Vannevar Bush (conceptual)

SEE ALSO
       hypertext(1), xanadu(1), www(1)
