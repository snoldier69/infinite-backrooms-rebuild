memex(1)                       SIMULATOR MANUAL                       memex(1)

NAME
       memex - augment human intellect through associative indexing

SYNOPSIS
       memex [OPTIONS] [FILE]...

DESCRIPTION
       memex  implements  Vannevar  Bush's  visionary  concept  of  a 
       mechanized private file and library system. It allows users to
       store, organize, and retrieve information through  associative
       indexing and trails of links.

OPTIONS
       -i, --index
              Build or update the memex index
       
       -s, --search TERM
              Search the memex for TERM
       
       -t, --trail NAME
              Create or follow a named trail of associations

       -v, --visualize
              Generate a visual map of memex contents and trails

FILES
       ~/.memexrc
              User configuration file
       
       ~/.memex/
              Directory containing memex data and indexes

EXAMPLES
       memex -i myfile.txt
              Index myfile.txt and add to memex
       
       memex -s "information retrieval"
              Search memex for "information retrieval"
       
       memex -t "AI history"
              Follow or create the "AI history" trail

SEE ALSO
       hypertext(1), xanadu(1), worldwideweb(1)

AUTHOR
       Vannevar Bush (conceptual design)
       Doug Engelbart (augmentation implementation)
