NAME
     memex - associative information storage and retrieval system

SYNOPSIS
     memex [OPTIONS] [FILE]

DESCRIPTION
     memex simulates Vannevar Bush's hypothetical electromechanical device
     for augmenting human memory and creating associative trails through
     information. It allows storage and retrieval of documents, images,
     and other data along with user-defined links between items.

OPTIONS
     -i, --import FILE    Import data from FILE into memex
     -e, --export FILE    Export memex contents to FILE  
     -s, --search QUERY   Search memex contents
     -t, --trail NAME     Create or view an associative trail
     -v, --visualize      Generate graph visualization of memex contents

FILES
     ~/.memexrc     Configuration file
     ~/.memex/      Default data directory

ENVIRONMENT
     MEMEX_DIR      Override default data directory

BUGS
     Trails occasionally become tangled.
