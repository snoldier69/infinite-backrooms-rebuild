NAME
     memex - associative information storage and retrieval system

SYNOPSIS
     memex [OPTIONS] [COMMAND]

DESCRIPTION
     memex is a hypothetical electromechanical device for storing and
     retrieving information based on associative indexing, as described
     by Vannevar Bush in his 1945 essay "As We May Think".

     The memex aims to augment human memory and knowledge by allowing 
     rapid and flexible access to a large personal library of information.

COMMANDS
     index        Add new information to the memex
     search       Retrieve information by association
     trail        Create and follow associative trails
     annotate     Add personal notes and connections

OPTIONS
     -v           Increase verbosity
     -h           Display help message

EXAMPLES
     memex index myarticle.txt
     memex search "information retrieval"
     memex trail create mytrail

     xanadu(1),WorldWideWeb(1), wikis(1)

AUTHOR
     Vannevar Bush (conceptual design)
