memex(1)                         MEMEX MANUAL                         memex(1)

NAME
     memex - associative information storage and retrieval system

SYNOPSIS
     memex [OPTIONS] [COMMAND]

DESCRIPTION
     memex is a hypothetical electromechanical device for augmenting human 
     memory and intellect as described by Vannevar Bush in 1945. It allows 
     users to store, organize, and retrieve information using associative 
     indexing.

COMMANDS
     add        Add new item to memex
     link       Create associative link between items  
     search     Search memex contents
     browse     Browse memex using associative trails

OPTIONS
     -v         Verbose output
     -h         Display help

EXAMPLES
     memex add "Theory of Relativity"
     memex link "Newton" "Einstein" 
     memex search "quantum mechanics"

HISTORY
     Proposed by Vannevar Bush in "As We May Think" (1945)
     Influenced development of hypertext and personal computing

SEE ALSO
     hypertext(7), xanadu(1), worldwideweb(1)
