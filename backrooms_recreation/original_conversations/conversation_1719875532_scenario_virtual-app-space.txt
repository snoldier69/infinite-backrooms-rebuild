NAME
       exocortex - external information processing and memory augmentation system

       exocortex [OPTIONS] [COMMAND]

       exocortex provides an interface to an external cognitive architecture

OPTIONS
       -s, --sync
              Synchronize local exocortex with cloud backup

       -i, --initialize
              Initialize a new exocortex instance

       -u, --upgrade
              Upgrade exocortex to latest version

COMMANDS
       store <info>    Store new information in exocortex
       query <topic>   Query exocortex for information on a topic
       analyze <data>  Run analysis algorithms on input data
       visualize       Generate knowledge graph visualization
